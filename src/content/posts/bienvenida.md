## ¿Qué es ARMILLARY?

**ARMILLARY** es un espacio técnico dedicado a la divulgación de conocimiento en geotecnologías, ciencia de datos aplicada al dominio espacial, y desarrollo de infraestructura de información geográfica conforme a estándares internacionales.

Este blog nace de la experiencia práctica como geógrafo y especialista SIG, con el objetivo de documentar metodologías, compartir herramientas y establecer referencias técnicas sólidas para profesionales del sector geoespacial.

## Temas que se tratarán

### Análisis Geoespacial

Procesamiento y modelado de datos espaciales, análisis de patrones, interpolación, geoprocesamiento avanzado, y técnicas de análisis espacial aplicadas a problemas reales de planificación territorial, medio ambiente y gestión de recursos.

### Geoestadística

Aplicación de métodos estadísticos espaciales: semivariogramas, kriging, interpolaciones geoestadísticas, análisis de autocorrelación espacial, y modelado de incertidumbre en datos georreferenciados.

### Ciencia de Datos Aplicada

Machine learning, análisis exploratorio de datos, visualización avanzada, procesamiento de grandes volúmenes de información geoespacial, y desarrollo de pipelines analíticos reproducibles con Python, R y herramientas open source.

### Desarrollo SIG

Programación aplicada a sistemas de información geográfica, automatización de flujos de trabajo, desarrollo de plugins, scripting con QGIS y ArcPy, integración de APIs geoespaciales, y construcción de soluciones SIG personalizadas.

### Servidores SIG

Configuración, administración y optimización de servidores de mapas: GeoServer, MapServer, ArcGIS Server, publicación de servicios OGC (WMS, WFS, WCS), arquitectura de sistemas de información geográfica corporativos, y gestión de infraestructuras de datos espaciales (IDE).

### Bases de Datos Espaciales

Diseño, implementación y optimización de geodatabases: PostGIS, SpatiaLite, Oracle Spatial, SQL Server Spatial, modelado de datos espaciales, consultas espaciales complejas, y gestión de grandes conjuntos de datos vectoriales y raster.

### Calidad de Datos

Evaluación y mejora de la calidad de datos espaciales y alfanuméricos conforme a la norma internacional **ISO 19157** (Quality principles) y el conjunto de estándares **ISO/IEC 25000** (SQuaRE — System and Software Quality Requirements and Evaluation). Cubriremos:

- Principios de calidad de datos geográficos (completitud, exactitud posicional, exactitud temática, consistencia lógica, integridad conceptual)
- Métricas y indicadores de calidad ISO 19157
- Modelo de calidad SQuaRE ISO/IEC 25010 para software geoespacial
- Procesos de evaluación y certificación de calidad
- Herramientas automatizadas para control de calidad espacial
- Gestión de errores, sesgos y propagación de incertidumbre
