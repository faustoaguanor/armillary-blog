## ¿Qué es ARMILLARY?

**ARMILLARY** es un espacio dedicado a la exploración y divulgación de temas en:

- **Análisis Geoespacial**: Procesamiento de datos espaciales, SIG, teledetección
- **Ciencia de Datos**: Machine learning, estadística, visualización
- **Desarrollo GIS**: Programación aplicada a sistemas de información geográfica

## ¿Qué encontrarás aquí?

Este blog nace de la necesidad de compartir conocimientos adquiridos a lo largo de años de trabajo como geógrafo y desarrollador. Aquí publicaré tutoriales, guías técnicas, y reflexiones sobre tecnología geoespacial.

### Python para GIS

Tutoriales prácticos sobre cómo usar Python para análisis geoespacial:

```python
import geopandas as gpd
import matplotlib.pyplot as plt

# Cargar datos espaciales
gdf = gpd.read_file("datos.shp")
gdf.plot()
plt.show()
```
